# Plan: Progressive Reading Word Count by Week

## Goal
Increase reading word count by 100 words every 7 completed app days (dayIndex-based), starting at 200–300 words for days 0–6.

## Word Count Formula
```
weekNumber  = Math.floor(dayIndex / 7) + 1   // 1, 2, 3, ...
min         = 200 + (weekNumber - 1) * 100   // 200, 300, 400, ...
max         = 300 + (weekNumber - 1) * 100   // 300, 400, 500, ...
```

| dayIndex | week | word count |
|---|---|---|
| 0–6   | 1 | 200–300 |
| 7–13  | 2 | 300–400 |
| 14–20 | 3 | 400–500 |
| 21–27 | 4 | 500–600 |
| 28–34 | 5 | 600–700 |
| …     | … | … |

No cap — grows indefinitely with `+100 words/week`.

**Bonus readings** (dayIndex ≥ 100,000) use the user's actual `currentDayIndex` for week calculation, so they always match the user's current level.

---

## Files to Change

### 1. `apps/web/lib/ai/constants.ts`

Convert `READING_CURATION_PROMPT` and `READING_FALLBACK_PROMPT` from string constants to functions accepting `(min: number, max: number)`. Word count references become interpolated.

```ts
// Before
export const READING_CURATION_PROMPT = `...Output EXACTLY 400-600 words...`;
export const READING_FALLBACK_PROMPT = `...body (400-600 words)...`;

// After
export function readingCurationPrompt(min: number, max: number): string {
  return `...Output EXACTLY ${min}-${max} words...`;
}
export function readingFallbackPrompt(min: number, max: number): string {
  return `...body (${min}-${max} words)...`;
}
```

### 2. `apps/web/lib/ai/reading.ts`

**Add helper** (before `generateReadingStream`):
```ts
function getWordCountRange(dayIndex: number): { min: number; max: number } {
  const weekNumber = Math.floor(dayIndex / 7) + 1;
  return {
    min: 200 + (weekNumber - 1) * 100,
    max: 300 + (weekNumber - 1) * 100,
  };
}
```

**Update `generateReadingStream` signature** — add optional `weekDayIndex` param:
```ts
export async function generateReadingStream(
  userId: string,
  dayIndex: number,
  onDelta: (delta: string) => void | Promise<void>,
  weekDayIndex?: number,  // for word-count week calculation; defaults to dayIndex
): Promise<GeneratedReading>
```

**Inside the function:**
- `const { min, max } = getWordCountRange(weekDayIndex ?? dayIndex);`
- Replace `READING_CURATION_PROMPT` → `readingCurationPrompt(min, max)` and update imports
- Replace `READING_FALLBACK_PROMPT` → `readingFallbackPrompt(min, max)`
- Update both user prompt strings: `body (${min}-${max} words)` (appears twice — curation and fallback paths)
- `maxOutputTokens: Math.max(1200, max * 2)` (scales token budget with word target)

**Day 0 special case is unchanged** — it streams the hardcoded welcome text and returns early before any of the above logic.
